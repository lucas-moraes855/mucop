Iniciar

Siga o passo a passo abaixo para começar a utilizar o framework em seus projetos.

Sobre

O framework Mucop foi criado com o objetivo de padronizar os estilos das aplicações desenvolvidas pelo departamento de TI da Intrafederativa, a fim de representar a identidade da marca Unimed e otimizar o tempo de desenvolvimento dos serviços.

Download e Instalação

Faça o download do framework clicando no botão abaixo, extraia o arquivo compactado e adicione o diretório completo em sua aplicação.Nele constará os arquivos CSS, JavaScript e fontes.

Download v1.0.0

Importação dos arquivos

Adicione a seguinte folha de estilos na tag <head> da página:

<link rel="stylesheet" href="_mucop/css/mucop.min.css">

Depois adicione a biblioteca jQuery e o script do framework no final da tag <body>:

<script src="_mucop/js/jquery-3.6.1.min.js"></script> <script src="_mucop/js/mucop.min.js"></script>

Tipografia

Foram definidas dois tipos de fontes nos estilos: Unimed Slab (títulos) e Trebuchet MS (demais textos). Para personalizar algum item específico utilizando a primeira fonte, será necessário utilizar uma das seguintes variáveis na propriedade font-family do CSS. Veja:

Unimed Slab Book

O Lorem Ipsum é um texto modelo --unislab-300

Unimed Slab Regular

O Lorem Ipsum é um texto modelo --unislab-400

Unimed Slab SemiBold

O Lorem Ipsum é um texto modelo --unislab-400

Unimed Slab Bold

O Lorem Ipsum é um texto modelo --unislab-500

Exemplo:

h1 { font-family: var(--unislab-500) }
